### コマンド

```sh
$ mkdir {lib, spec}
$ touch lib/stack.rb spec/spec_helper.rb spec/stack_spec.rb
```

```ruby
# lib/stack.rb

end
```

```ruby
# spec/spec_helper.rb

```

```ruby
# spec/stack_spec.rb
require 'spec_helper'

RSpec.describe Stack do
end
```

## No.1 Stack の仕様を考える

1. push 出来る?
2. pop 出来る?
3. length を取得出来る?
4. empty? かどうかを尋ねられる？

## No.2 spec を書いて実行する

```ruby
# spec/stack_spec.rb
require 'spec_helper'

RSpec.describe Stack do
  let(:stack) { Stack.new }

  describe "#push" do
    subject { stack.push 1 }
    it { expect(subject).to eq stack }
  end
end
```
